# MIPS Emulator
This is an emulator for the MIPS architecture, capable of running executables generated by my assembler (https://github.com/pclava/mips_assembler).
The emulator only supports a subset of the instructions supported by the assembler (e.g., excluding floating-point instructions). 

The goals of this project were to learn C++ and to learn how emulators work. I plan to implement rudimentary support for graphics and other I/O.

## Usage
The emulator expects an executable generated by the assembler with the following format:
```
* Header
  - 4 bytes: Text size (in bytes)
  - 4 bytes: Data size (in bytes)
  - 4 bytes: Program entry (32-bit memory address)
* Text segment
* Data segment
```

The emulator expects an executable and, optionally, arguments for the `main` function, if the program has one:

`$ ./MIPSEmulator <executable> <arg1, ...>`
